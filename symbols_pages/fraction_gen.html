How it works

This tool takes a fraction such as 3/4 and turns it into a Unicode-style fraction using superscript and subscript digits.

It can also use the numerator and denominator boxes directly, which is handy if you want cleaner validation in the JavaScript.

Some common fractions may already exist as single Unicode symbols, while others will be built from numerator, fraction slash, and denominator. Unicode loves consistency almost as much as a cat loves bath time.
