Why this page is useful

Fast classroom reference

It is much easier to identify and compare circuit symbols when they sit in one filterable visual set instead of being buried in separate documents.

Booklet differences made obvious

The current and older booklet sets are not identical, so this page helps you see what changed without playing spot-the-diode in a PDF.

Actually reusable

Each symbol can be downloaded as SVG, which is far more useful for worksheets, slides, and websites than a fuzzy screenshot from a scanned page.
