How it works

This tool converts ordinary Latin letters into Unicode script and bold script characters.

It also creates mixed styles, including alternating script and bold script, plus alternating capital and lowercase patterns for display text.

Some platforms handle fancy Unicode better than others. Unicode is brilliant right up until it decides to be an absolute goblin.
