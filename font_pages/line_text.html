Unicode Underline and Strikethrough Generator

This page is set up for Unicode underline and strikethrough styles, including mixed line effects, alternating caps versions, and bold or italic variants where Unicode supports them.
Quick examples

How it works

Most of these styles are created with Unicode combining marks. That means the underline or strike is layered onto each letter rather than drawn with CSS.

Bold and italic underline or strike versions are built by combining mathematical bold or italic Unicode letters with those same line marks.

If you keep copying the outcome and pasting back into the generator, you will create some funky designs.

Some platforms render combining marks beautifully. Others behave like they were coded on an Apple 2e.
